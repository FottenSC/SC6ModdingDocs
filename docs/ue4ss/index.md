# UE4SS Framework

UE4SS (Unreal Engine 4/5 Scripting System) injects into the game process, exposes the Unreal
reflection system (UObjects, UFunctions, UProperties) to Lua, and lets you hook engine-level functions.

- [Lua API Overview](lua-api.md) — globals, object access, and common patterns.
- [Hooks & Events](hooks.md) — `RegisterHook`, `NotifyOnNewObject`, and friends.
